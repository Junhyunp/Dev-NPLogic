# Assets 폴더

이 폴더에 로고 이미지 파일을 넣으세요.

## 이미지 파일 추가 방법

### 1. 이미지 파일을 이 폴더에 복사
예: `logo.png`, `logo.svg` 등

### 2. `.csproj` 파일에 등록
파일을 추가한 후 자동으로 프로젝트에 포함됩니다.

### 3. XAML에서 사용
```xml
<Image Source="/Assets/logo.png" Width="32" Height="32"/>
```

## 권장 이미지 형식
- **PNG**: 투명 배경 지원, 일반적으로 사용
- **SVG**: 벡터 이미지 (WPF에서는 추가 라이브러리 필요)
- **ICO**: 아이콘 파일

## 권장 크기
- **헤더 로고**: 128x128px ~ 256x256px
- **작은 아이콘**: 32x32px, 48x48px
- **창 아이콘**: 16x16px, 32x32px
