# NPlogic - 주요 개발 범위

## 🗂️ 주요 개발 범위

### 1. 사용자 인증 및 권한 관리

- Supabase Auth 연동 (로그인/회원가입)
- 역할 기반 권한 관리 (PM, 평가자, 관리자)
- 세션 관리 및 자동 로그인
- 역할별 화면 접근 제어

### 2. 데이터 업로드 및 관리

- 엑셀 데이터 디스크 업로드 (채권 기초 데이터 일괄 업로드)
- 자동 컬럼 매핑 시스템
- 등기부등본 PDF 대량 업로드 (500~3,000개)
- Supabase Storage 연동 및 파일 관리
- 파일 목록 조회, 삭제, 재업로드 기능
- 업로드 진행상황 표시

### 3. OCR 시스템 연동

- 클라이언트 제공 Python OCR 코드 C# 통합
- 대량 처리 최적화 (배치 처리, 멀티프로세싱)
- OCR 처리 진행상황 모니터링
- 오류 처리 및 재처리 기능
- 처리 결과 검증 시스템

### 4. 데이터 표시 및 편집

- 기초 데이터 화면 (자동 입력 필드 + 평가자 입력 필드)
- 등기부 정보 화면 (OCR 추출 데이터 구조화 표시)
- 권리 분석 정보 화면
- 데이터 실시간 수정 기능
- 데이터 병합 기능
- 데이터 삭제 기능
- 리스트 관리 (추가, 수정, 삭제)

### 5. 데이터 처리 및 계산

- 자동 데이터 매칭 (데이터 디스크 ↔ 등기부등본)
- 계산 수식 자동 적용 시스템
- 데이터 검증 (필수 항목 체크)
- 오류 데이터 감지 및 표시
- 데이터 무결성 검증

### 6. Excel 출력 시스템

- 여러 시트 데이터 통합 ("비핵심 전체" 형태)
- Excel 파일 생성 (다중 시트)
- 수식 자동 적용
- 스타일 및 포맷 적용
- 다중 출력 옵션 (전체/선택/필터)
- EPPlus 또는 ClosedXML 라이브러리 활용

### 7. 지도 연동 (디스코 맵)

- 클라이언트 제공 HTML 지도 코드 WebView 통합
- 지도 표시 (지적도, 위성지도, 로드뷰)
- 토지이용계획 표시
- 건축물 대장 표시
- 물건 정보 시각화 (마커, 매매가, 토지/건물 정보)
- WebBrowser Control 또는 CefSharp 활용

### 8. 유사 물건 추천

- 클라이언트 제공 추천 알고리즘 C# 통합
- 과거 경매 데이터 Supabase 연동
- 추천 결과 표시
- 유사도 기반 정렬
- 사례 지도 기능 (추천 물건 지도 표시)

### 9. 통계 분석

- Supabase 쿼리 기반 통계 처리
- 차트 시각화 (타입별, 기간별, 지역별)
- 통계 Excel 출력
- LiveCharts 또는 OxyPlot 활용
- 대시보드 통계 카드

### 10. 물건 관리

- 물건 목록 화면 (검색, 필터, 정렬)
- 페이지네이션
- 물건 상세 화면 (탭 구조)
  - 기초 데이터 탭
  - 등기부 정보 탭
  - 권리 분석 탭
  - 평가 시트 탭
  - 지도 탭
- 대시보드 (통계 카드, 차트, 빠른 작업)

### 11. 관리자 기능

- 사용자 관리 (계정 생성/수정/삭제)
- 역할 할당 및 권한 관리
- 작업 이력 관리 (수정 이력 추적, 되돌리기)
- 설정 관리 (계산 수식, 데이터 매핑, 시스템 환경)
- 시스템 로그 조회

### 12. UI/UX 디자인 및 구현

- 클라이언트 레퍼런스 기반 디자인 시스템 구축 (`비핵심 프로그램화_v6-1.xlsx`)
- 차분하고 딱딱한 엑셀 스타일 UI
- 색상 시스템 (베이스: 딥네이비, 표: 밝은 블루그레이)
- 역할별 최적화 화면 (PM, 평가자, 관리자)
- 일관된 UI 컴포넌트 시스템

### 13. 데이터베이스 설계 및 구축

- Supabase PostgreSQL 스키마 설계
- Row Level Security (RLS) 설정
- Storage 버킷 구성 (등기부등본 PDF 저장)
- 인덱싱 및 쿼리 최적화
- 데이터 마이그레이션 스크립트

### 14. 시스템 통합 및 배포

- C# Windows 애플리케이션 패키징 (Windows .exe)
- ClickOnce 또는 Inno Setup 설치 프로그램 생성
- 자동 업데이트 시스템 (Auto-Updater.NET)
- 설치 매뉴얼 작성
- 사용자 교육 자료 작성
