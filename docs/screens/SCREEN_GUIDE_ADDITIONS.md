## 구현 우선순위

1. **Phase 1**: 차주개요, 담보총괄 (데이터 요약 화면)
2. **Phase 2**: Loan 상세, 현금흐름 집계 (재무 계산 화면)
3. **Phase 3**: 경매일정 (비용 시뮬레이션)
4. **Phase 4**: XNPV 비교, 회생개요 (고급 분석)
5. **Phase 5**: Tool Box (참조 데이터 관리)

- cumulative_cash_flow, note

### 회생 (Restructuring)
- restructuring_id, borrower_id (FK), case_number
- court, opening_date, approval_date
- repayment_period_years
