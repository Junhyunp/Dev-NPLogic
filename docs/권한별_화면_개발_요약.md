# 권한별 화면 시스템 개발 완료

## 무엇을 했나요?

사용자 역할(관리자/PM/평가자)에 따라 **로그인하면 각자 다른 화면**을 보여주도록 만들었습니다.

## 역할별 화면

- **관리자**: 왼쪽에 전체 프로그램 목록, 오른쪽에 선택한 프로그램의 물건 목록
- **PM**: 자기가 담당하는 프로그램의 물건 목록만 바로 보임
- **평가자**: 자기에게 배정된 물건만 보임 + 진행률 표시

## 추가된 관리 기능

- 관리자가 프로그램을 만들고 PM을 지정할 수 있는 화면
- 사용자 정보에 소속팀, 직책, 회계법인 정보 추가
