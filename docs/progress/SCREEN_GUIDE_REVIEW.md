```
1. 신문공고료 (경매회수 * 단가)
2. 현황조사수수료 (감정건수 * 단가)
3. 경매매각수수료 (산식, 매각액기준)
4. 감정수수료 (산식, 감정액기준 - 복잡한 구간별 계산)
5. 송달수수료 ((이해관계인수 + 3) × 10회분)
6. 경매신청 등록/교육세 (산식, 경매신청액)
7. 기타비용
부대비용율 가산: 10%

경매예납금액 회수액 추정:
- 실제납입한 경매예납금액
- 예납금액수수료 회수비율 (90%)

3자경매/중복경매:
- 3자선행경매신청, 3자경매신청금액
- 중복경매신청, 중복경매신청금액
- 각각에 대한 비용 계산

Lead Time 로직:
- 차주사망/개회: 6개월
- 차주거주:
  - 주택담보대출: max((최종이수일+9)-adate, 0)
  - 주택담보대출 외: max((최종이수일+8)-adate, 0)
```
