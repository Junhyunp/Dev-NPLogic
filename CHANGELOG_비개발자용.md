## 📋 이번 업데이트에서 바뀐 것들

### ✅ 해결된 문제 (피드백 반영)

#### 1. 로그인 화면 개선
- **이전**: 로그인 화면에 불필요한 남색 테두리가 있었음
- **이후**: 테두리 제거하여 깔끔한 디자인으로 변경

#### 2. 대시보드 메뉴 구조 변경
- **이전**: 메뉴가 "홈, 비핵심, 등기부등본, 권리분석, 기초데이터, 마감"으로 제대로 구성되어 있지 않았음
- **이후**: 요청하신 대로 내부 탭 메뉴 구조로 변경 완료

#### 3. 대시보드 필터링 기능 추가
- **새 기능**: 물건 목록을 상태별로 필터링할 수 있음
  - 전체 / 대기 / 진행 / 완료 상태로 분류하여 볼 수 있음

#### 4. 차주번호/차주명 클릭 시 동작 수정
- **이전**: 차주번호나 차주명을 클릭하면 엉뚱한 화면으로 이동하거나, 심지어 차주가 삭제되는 오류 발생
- **이후**: 클릭하면 정상적으로 "홈(전체)" 화면으로 이동

#### 5. 탭 이동 시 저장 안내 간소화
- **이전**: 비핵심_전체, 차주개요, 론 등 탭을 이동할 때마다 저장 안내 팝업이 반복적으로 나타나서 번거로웠음
- **이후**: 탭 전환 시 저장 안내 팝업 없이 바로 전환됨

#### 6. 담보 물건 지도 화면 연결
- **문의 사항**: 기존에 있던 지도 화면이 어디로 갔는지
- **해결**: 비핵심 화면의 "담보 물건" 탭에서 지도를 볼 수 있도록 연결 완료

---

### 🆕 새로 추가된 기능

#### 1. 비핵심 탭 상태 유지
- **새 기능**: 비핵심 화면에서 상단 탭(R-002_01 등)을 닫아도, 다른 메뉴(홈, 마감 등) 갔다가 돌아오면 이전 상태가 그대로 유지됨
- **장점**: 매번 탭을 다시 열 필요 없어 작업 효율 향상

#### 2. 홈에서 물건 선택 시 자동 탭 전환
- **새 기능**: 홈 화면에서 물건을 선택하면 자동으로 비핵심 탭으로 전환됨
- **장점**: 클릭 횟수 감소, 업무 흐름 개선

#### 3. 비핵심 화면 레이아웃 변경
- **변경 내용**: 탭 리스트가 화면 전체 너비로 맨 위에 배치되고, 검색/툴박스 영역이 그 아래에 위치하도록 변경
- **장점**: 더 많은 탭을 한눈에 볼 수 있음

#### 4. 세션 만료 처리 개선
- **새 기능**: 로그인 세션이 만료되면 자동으로 감지하여 로그인 화면으로 안내
- **장점**: 갑자기 오류가 나는 대신 친절한 안내 메시지 제공
