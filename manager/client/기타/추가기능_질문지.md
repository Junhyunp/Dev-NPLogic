## 질문 1: L-001 인터림 파일 업로드 (상계회수 금액 자동 매핑)

### 배경
인터림 파일에서 상계회수 금액 등의 데이터를 자동으로 시스템에 매핑하는 기능 구현이 필요합니다.
현재 기존 Excel 업로드 파이프라인(`ExcelService`, `SheetMappingConfig`, `DataUploadViewModel`)이 이미 구축되어 있어, 
이 구조를 확장하여 인터림 파일도 처리할 예정입니다.

### 질문 항목

**1. 인터림 파일의 최종 컬럼 구조를 공유해주실 수 있나요?**
   - 컬럼명 (한글/영문)
   - 컬럼 순서
   - 데이터 타입 (숫자, 텍스트, 날짜 등)
   - 샘플 파일이 있으면 더욱 좋습니다

**2. 물건/차주 매칭에 사용할 키는 무엇인가요?**
   - 예시:
     - [ ] 차주번호 단독
     - [ ] 차주번호 + 채권번호 조합
     - [ ] 담보번호
     - [ ] 기타: _______________

**3. 상계회수 금액 외에 추가로 매핑해야 할 필드가 있나요?**
   - 현재 시스템에서 관리 중인 필드:
     - `offset_recovery` (상계회수)
     - `interring` (인터링)
   - 추가 필요 필드가 있으면 기재해주세요: _______________

## 질문 2: L-002 당해세 자동계산 수식

### 배경

현재는 etax korea 링크를 통해 수동으로 조회 후 입력하는 방식이 구현되어 있습니다 (X-001 완료).
자동계산 기능 구현을 위해 추가 정보가 필요합니다.

### 질문 항목

**1. "도시지역" 판단 기준은 무엇인가요?**
   - 예시:
     - [ ] 특정 행정구역명 (예: 서울, 부산 등)
     - [ ] 용도지역 (도시지역/비도시지역)
     - [ ] 특정 코드 (예: 지역코드 앞자리 기준)
     - [ ] 기타: _______________

**2. 현재 etax korea 수동 입력 방식으로 충분한지, 자동계산이 반드시 필요한지 우선순위를 알려주세요.**
   - [ ] 자동계산 기능 필수 (수동 입력과 병행)
   - [ ] 현재 수동 입력으로 충분 (자동계산은 후순위)

### 기능 상세

1. **X-002: 대법원 경매 링크**
   - 선순위 화면에 "대법원 경매 바로가기" 버튼 추가
   - URL: https://www.courtauction.go.kr

2. **L-003: 소액임차보증금 자동 조회**
   - 지역(서울/수도권/광역시/기타) + 근저당설정일 선택
   - "자동 조회" 버튼 클릭 시 해당 시점의 소액임차보증금 기준액 조회
   - 조회 결과를 DD금액에 자동 반영

3. **N-001: 키보드 탐색**
   - `Ctrl + ←`: 이전 물건
   - `Ctrl + →`: 다음 물건
   - `Ctrl + ↑`: 첫 번째 물건
   - `Ctrl + ↓`: 마지막 물건
   - 이동 시 변경사항 자동 저장

4. **N-002: 연속 작업 UI**
   - 물건 상세 화면 헤더에 `◀ [2/15] ▶` 형태의 네비게이션 UI 추가
   - 버튼 클릭으로 이전/다음 물건 이동 가능
