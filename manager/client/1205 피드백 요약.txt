=== 1205 미팅 피드백 요약 (개발 반영 사항) ===

[ UI/디자인 ]
1. 로그인 화면 톤 변경: 현재 너무 진함 → 기존 남색 톤으로 변경
2. Header UI 수정:
   - 엠필로지 로고 워딩 사용
   - 헤더 남색 배경 + 알림/설정/나가기 아이콘 흰색 톤
   - 화면 테두리를 비슷한 톤으로 둘러싸기 (엑셀 스타일처럼)
3. 첫 번째 화면 ↔ 두 번째 화면 톤 통일 (같은 프로그램처럼 연결성 있게)

[ 대시보드 구조 ]
4. "메뉴창" 텍스트 제거
5. 대시보드 구성: "진행 중인 프로그램" | "프로그램명" 두 가지 카테고리로 구분
6. 두 영역 간 width 크기 조절 가능 (마우스 드래그로)
7. 두 영역 모두 닫기/열기 가능하도록
8. 상단 기능 메뉴(홈/비핵심/등기부/권리분석/기초데이터/마감) → 프로그램명 내부로 위치 변경
9. 대시보드 컬럼별 진행율 퍼센테이지로 표시 (별도 테이블 X, 약정서 몇%, 보증서 몇% 형태)

[ 화면 이동/네비게이션 ]
10. 진행 중인 프로그램 클릭 → 오른쪽 프로그램명에 해당 프로그램 데이터 출력
11. 프로그램명(대시보드)의 차주번호 클릭 → 해당 차주번호의 '비핵심' 대시보드로 이동
    (이때 홈/비핵심/등기부/권리분석/기초데이터/마감 Header는 유지, 내부 테이블만 변경)
12. 페이지/탭 이동: 버튼 또는 단축키(Shift+Tab, Alt+방향키 등)로 다음 페이지(탭) 이동
13. 비핵심 대시보드 ↔ 프로그램명 대시보드 간 상하좌우 이동 자유롭게

[ 비핵심 내부 구조 ]
14. 비핵심 내부 탭 구성: 차주 개요 / 론 / 담보 총괄 / 담보 물건 / 선순위 평가 / 경공매 / 회생개요 / 현금 흐름 집계 / NPB
15. 물건 번호 탭처럼 나열 (PDF 여러 개 띄워놓은 것처럼)
16. 툴박스는 옆(사이드)으로 배치
17. 선순위만, 평가만 등 기능별로 먼저 쭉 작업할 수 있어야 함

[ 롤업 기능 ]
18. 롤업 버튼 클릭 → 전체 화면(비핵심 전체)으로 이어짐
19. 각 탭에서 수정 시 전체 화면에 즉각 반영
20. 롤업 창은 수정 중에도 닫히면 안 됨 (계속 띄워져 있어야 함)

[ 권한별 화면 ]
21. 관리자/유안코: 전체 풀 다 보임
22. 회계법인 PM: 담당 전체 보임
23. 평가자: 자기가 assign된 것만 보임

[ 불필요 기능 ]
24. 물건 목록 기능 필요 없음
25. 통제 기능 필요 없음

[ 데이터 업로드 ]
26. 기존 데이터 업로드 화면 삭제 → 비핵심 기초데이터 화면에서 데이터 디스크 업로드
27. 업로드 버튼 + 업로드 일자 표시

[ 설계 파일 업데이트 (버전 7.1) ]
28. 물건에 OPB 컬럼 추가
29. 자본소득률 → 상가/아파트형공장 상권 데이터 추가 (체크박스 추가, 평가사 4번/5번만 해당)
30. 인터링, 상계회수 항목 추가 (경매 일정 맨 밑에)
31. 계좌과 대출 과목에서 C 하나 삭제

[ 후순위 (시간 여유 있으면) ]
32. 소상공인 상권 데이터 지도 연동 검토
    - 물건 위치 자동 표시 + 가까운 상권 클릭 시 표 출력
    - 좌표 받기 어려움 → 구현 방법 추가 검토 필요

[ 데이터 관련 ]
33. 경매 데이터 추출 완료되면 전달 예정
34. 일부 주소(금화 주차장 등) 좌표 변환 불가 → 크리티컬하지 않음, 나중에 처리
35. 주소 앞뒤 공란 제거 체크 필요
