# NPLogic

WPF 기반 Windows 데스크톱 애플리케이션

## 요구사항

- .NET 8.0 SDK
- Visual Studio 2022 (또는 그 이상)

## 빌드 방법

```bash
dotnet build
```

## 실행 방법

```bash
dotnet run
```

## 프로젝트 구조

- `App.xaml` / `App.xaml.cs` - 애플리케이션 진입점
- `MainWindow.xaml` / `MainWindow.xaml.cs` - 메인 윈도우
- `NPLogic.csproj` - 프로젝트 파일
